# Data Visualization and Storytelling

## Objective
Create visualizations that communicate meaningful business insights using Power BI.

## Tool Used
Power BI

## Dataset
Superstore Sales Dataset

## Dashboard Components
- Sales Trend by Year
- Sales by Region
- Sales by Category
- Top States by Sales
- Top Products by Sales
- Category and Segment Analysis

## Key Insights
- Technology category generated the highest sales.
- West region achieved the highest revenue.
- California contributed the highest sales.
- Sales increased significantly from 2016 to 2018.
- Consumer segment generated the largest share of revenue.
